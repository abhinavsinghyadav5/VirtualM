VirtualM – Virtual Mouse & Keyboard:

A Computer Vision based Virtual Mouse and Keyboard system that allows users to control their computer using hand gestures. This project uses OpenCV and MediaPipe to detect hand landmarks in real-time and map gestures to mouse and keyboard actions, enabling hands-free computer interaction.

Features:

•	 Real-time hand gesture recognition
•	 Control mouse movement using finger tracking
•	 Perform left click and right click gestures
•	 Scroll functionality using specific gestures
•	 Virtual keyboard typing using hand landmarks
•	 Optimized frame processing for smooth performance
•	 Accurate hand landmark detection using MediaPipe

Tech Stack:

•	Python
•	OpenCV
•	MediaPipe
•	NumPy

How It Works:

1.	The webcam captures live video input.
2.	MediaPipe Hand Tracking detects hand landmarks.
3.	The program analyzes finger positions and distances.
4.	Specific gestures are mapped to mouse and keyboard actions.
5.	OpenCV processes frames in real time to ensure smooth interaction.
